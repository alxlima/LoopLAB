Social profiles

  Este é um modelo basico que pude aprender de uma landing page de uma página, como se fosse um site de tipo midia social.
  Neste exemplo uso blocos féxiveis com alinhamento de textos e imagens responsivas, incluido formulários no formato flex mobal, incluindo um nav bar para 
  para explorar conteúdos por seções tambem adicionei imagens de backgroup transparentes em 2 plano com os objetos em sobreposição.
  Ao final da pagina no roda pé criei um botão que abrirá um pequeno modal tendo como exemplos alguns campos de um formulario de cadatro.
